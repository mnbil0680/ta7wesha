## Preferred Design System (Recommended)

### Design Language

* **Material Design 3 (Material You)**

### UI Framework

* **Flutter Material 3** (if using Flutter)
* **Jetpack Compose Material 3** (Android)
* **SwiftUI** with Material-inspired design (iOS)

### Design Principles

* Clean
* Minimal
* Modern
* Financial Dashboard Style
* Data-focused
* Accessibility-first

### Color Palette

* **Primary:** Blue (#2563EB)
* **Success:** Green (#16A34A)
* **Warning:** Orange (#F59E0B)
* **Danger:** Red (#DC2626)
* **Background:** White / Light Gray
* **Dark Mode:** Full support

### Typography

* **Inter** (Recommended)
* SF Pro (iOS)
* Roboto (Android)

### Icons

* Material Symbols Rounded

### Charts

* Interactive Line Charts
* Bar Charts
* Pie/Donut Charts
* Candlestick Charts (Markets)
* Area Charts

### Navigation

* Bottom Navigation Bar (5 tabs)
* Top App Bar
* FAB for Quick Add Transaction

### Overall Style

* Banking/FinTech UI
* Card-based Layout
* Soft Shadows
* Rounded Corners (12–16px)
* Smooth Animations
* Responsive Design
* Light & Dark Themes

> **Best choice overall:** **Material Design 3 (Material You)**, as it is mature, highly scalable, accessible, and particularly well suited for modern finance and banking applications.
